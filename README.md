# Ebook Landing Page

A responsive landing page built with **HTML**, **CSS**, and **Bootstrap 4**, designed for an Amazon ebook publishing and formatting service. The page includes a structured hero section, service details, pricing tabs, and a contact form, showcasing clean UI layout and front-end development fundamentals.
